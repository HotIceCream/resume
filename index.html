Опыт работы и достижения:

Июнь 2011г. – настоящее время. – инженер-программист, Инжиниринговый центр НИЯУ МИФИ. За время работы на данном месте я занимался разработкой приложений для корпоративного использования (защищены NDA) и для внутреннего использования:

  • WhiteLynx – во время работы над данным проектом я занимался разработкой защищенного протокола для VoIP телефонии, реализации нативной кросплатформенной библиотеки, основанной на WebRTC, которая использовалась в iOS и Android приложениях. А так же разработка Android приложения, позволяющего производить VoIP звонки и обмениваться мгновенными сообщениями и файлами.
  • YMessanger – приложение для обмена текстовыми сообщениями, основанное на решение TextSecure. За время работы над данным приложением мне удалось показать свои навыки в разработке пользовательского интерфейса, оптимизации производительности приложения и потребления памяти.
  • Sweedate – при работе над данным проектом я взаимодействовал с API популярных соцсетей (Vk, Ok, Facebook, Instagramm). Взаимодействовал с серверной частью нашего приложения (использовал RxJava, Retrofit, Gson, OkHttp, Glide). Писал собственные ViewGroup для расположения элементов согласно дизайну. Имел опыт анимирования элементов интерфейса.
  • Другие приложения – при работе над приложениями для заказчиков мне приходилось решать сложные задачи по взаимодействию отдельных приложений, входящих в общую систему, взаимодействию с сервером, разработке для PC (PyQt).
